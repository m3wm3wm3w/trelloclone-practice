# Как записать демонстрацию проекта

## Требования

- **Формат:** GIF, видео или asciinema
- **Длительность:** До 2 минут
- **Содержание:** Основной сценарий использования с результатом

## Рекомендуемые инструменты

### Windows - ScreenToGif (лучший выбор)

**Скачать:** https://www.screentogif.com/

**Почему ScreenToGif:**
- Бесплатный и легкий
- Запись напрямую в GIF
- Встроенный редактор
- Можно удалять кадры, добавлять текст
- Готовый GIF для вставки в отчет

**Как использовать:**
1. Запустите ScreenToGif
2. Выберите "Recorder"
3. Настройте область записи (окно браузера)
4. Нажмите Record (F7)
5. Выполните сценарий
6. Нажмите Stop (F8)
7. В редакторе можно обрезать, удалить кадры
8. File → Save as → GIF
9. Настройте качество (рекомендуется 15-20 FPS для меньшего размера)

### Альтернатива - Loom (видео)

**Сайт:** https://www.loom.com/

**Почему Loom:**
- Простая запись с экрана
- Автоматический upload
- Получаете ссылку для вставки
- Можно встроить в отчет

**Как использовать:**
1. Зарегистрируйтесь на loom.com
2. Установите расширение для браузера или desktop app
3. Нажмите Start Recording
4. Выберите "Screen only"
5. Выполните сценарий
6. Нажмите Stop
7. Видео автоматически загрузится
8. Скопируйте ссылку для отчета

## Сценарий демонстрации (90-120 секунд)

### Подготовка
1. Очистите cookie браузера для чистого старта
2. Откройте деплой URL
3. Приготовьте второе окно/вкладку для демонстрации real-time

### Сценарий записи

**0:00-0:15 - Регистрация (15 сек)**
- Показать главную страницу
- Нажать "Sign Up"
- Заполнить форму (Demo / User / demo@test.com / demo123)
- Нажать "Sign Up"

**0:15-0:30 - Создание доски (15 сек)**
- На странице boards нажать "Create new board"
- Ввести "Project Demo"
- Нажать "Create"
- Показать пустую доску

**0:30-0:50 - Добавление списков (20 сек)**
- Нажать "+ Add another list"
- Ввести "To Do" → Enter
- Добавить "In Progress" → Enter
- Добавить "Done" → Enter

**0:50-1:10 - Создание карточек (20 сек)**
- В "To Do": добавить "Task 1" → Enter
- В "To Do": добавить "Task 2" → Enter
- В "In Progress": добавить "Current task" → Enter
- В "Done": добавить "Completed" → Enter

**1:10-1:25 - Редактирование карточки (15 сек)**
- Кликнуть на "Task 1"
- Изменить название на "Updated Task"
- Добавить описание "This is a test description"
- Нажать "Save"
- Показать обновленную карточку

**1:25-1:50 - Real-time демонстрация (25 сек)**
- Открыть ту же доску в новой вкладке/окне (расположить рядом)
- В первой вкладке создать новую карточку "Real-time test"
- Показать как карточка появляется во второй вкладке
- Удалить карточку в первой вкладке (показать модальное окно подтверждения)
- Показать как она исчезает во второй

**1:50-2:00 - Завершение (10 сек)**
- Показать список online пользователей
- Показать финальный вид доски со всеми элементами
- (опционально) Быстро показать главную страницу со списком досок

## Советы по записи

**До записи:**
1. Закройте лишние вкладки и программы
2. Отключите уведомления Windows
3. Увеличьте масштаб браузера до 100-110% для читаемости
4. Проверьте что интернет стабилен
5. Потренируйтесь выполнить сценарий 1-2 раза

**Во время записи:**
1. Двигайте мышь плавно
2. Делайте паузы 1-2 секунды после действий
3. Показывайте результаты действий
4. Не спешите - лучше 2 минуты качественной записи

**После записи:**
1. Просмотрите полностью
2. Удалите начало (до появления страницы) и конец
3. Убедитесь что всё видно и читаемо
4. Проверьте размер файла (для GIF рекомендуется <10 МБ)

## Оптимизация GIF

Если GIF слишком большой:

**В ScreenToGif:**
1. Options → Reduce Framerate → 15 FPS
2. Resize → 80% от оригинала
3. Delete → Remove duplicates
4. Save as → Выбрать "Higher compression"

**Онлайн сервисы:**
- https://ezgif.com/optimize (сжатие GIF)
- https://gifcompressor.com/ (уменьшение размера)

## Где разместить

### Для отчета Word/PDF:
- Вставить GIF напрямую в документ
- Или скриншоты ключевых моментов + ссылка на видео

### Для GitHub:
```markdown
![Demo](demo.gif)
```

### Loom видео:
```markdown
[Демонстрация проекта](https://www.loom.com/share/your-video-id)
```

## Альтернативный формат - Скриншоты + описание

Если не получается записать:

1. **Скриншот 1:** Страница регистрации
2. **Скриншот 2:** Главная страница с досками
3. **Скриншот 3:** Доска с списками и карточками
4. **Скриншот 4:** Модальное окно редактирования карточки
5. **Скриншот 5:** Модальное окно подтверждения удаления
6. **Скриншот 6:** Два окна с real-time синхронизацией

К каждому скриншоту краткое описание действия и результата.

## Проверка перед отправкой

- [ ] Длительность до 2 минут
- [ ] Показаны основные функции
- [ ] Видна real-time синхронизация
- [ ] Качество позволяет читать текст
- [ ] Размер файла разумный (<20 МБ)
- [ ] Нет личной информации на экране
- [ ] Демо показывает рабочий функционал

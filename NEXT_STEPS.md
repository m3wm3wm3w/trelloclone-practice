## 📋 Осталось сделать (по порядку)

### Шаг 1: Деплой Backend на Render (30 минут)

**Инструкция:** Откройте `DEPLOYMENT.md` → Шаг 1

**Кратко:**
1. Зайти на render.com через GitHub
2. New Web Service → выбрать репозиторий
3. Настроить: Build: `npm install`, Start: `node server/index.js`
4. Добавить environment variables (MONGODB_URI, JWT_SECRET, etc)
5. Нажать Create и дождаться деплоя
6. **Скопировать URL backend** (понадобится для следующего шага)

**Результат:** Backend работает на `https://your-app.onrender.com`

---

### Шаг 2: Деплой Frontend на Vercel (20 минут)

**Инструкция:** Откройте `DEPLOYMENT.md` → Шаг 2

**Кратко:**
1. Зайти на vercel.com через GitHub
2. New Project → выбрать репозиторий
3. **Важно:** Root Directory → `client`
4. Добавить environment variables:
   - REACT_APP_API_URL = (URL backend из шага 1)
   - REACT_APP_SOCKET_URL = (URL backend из шага 1)
5. Нажать Deploy
6. **Скопировать URL frontend**

**Результат:** Frontend работает на `https://your-app.vercel.app`

---

### Шаг 3: Обновить CLIENT_URL на Backend (5 минут)

**Инструкция:** Откройте `DEPLOYMENT.md` → Шаг 3

**Кратко:**
1. Вернуться в Render → ваш Web Service → Environment
2. Обновить CLIENT_URL на URL frontend из шага 2
3. Сохранить (сервис автоматически перезапустится)

**Результат:** CORS настроен, приложение полностью работает

---

### Шаг 4: Проверить работу (10 минут)

**Что проверить:**
1. Открыть Vercel URL в инкогнито режиме
2. Зарегистрировать нового пользователя
3. Создать доску
4. Добавить список и карточку
5. Открыть в двух вкладках - проверить real-time
6. Если всё работает - отлично! Если нет - см. Troubleshooting в DEPLOYMENT.md

**Результат:** Рабочее приложение в production

### Шаг 6: Настроить Code Climate (15 минут)

**Инструкция:** Откройте `CODE_CLIMATE_SETUP.md`

**Кратко:**
1. Зайти на codeclimate.com через GitHub
2. Add repository → выбрать trelloclone-practice
3. Дождаться анализа (3-5 минут)
4. Repo Settings → Badges → скопировать Maintainability Badge
5. Вставить в README.md (заменить placeholder)
6. Закоммитить и запушить

**Результат:** Бейдж Code Climate в README

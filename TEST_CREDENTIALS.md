## Инструкция по тестированию

### 1. Регистрация и вход
1. Откройте приложение
2. Нажмите "Sign Up" если хотите создать новый аккаунт
3. Или используйте существующие credentials выше

### 2. Создание доски
1. На главной странице нажмите "Create new board"
2. Введите название доски
3. Нажмите "Create"

### 3. Создание списков и карточек
1. Откройте доску
2. Нажмите "+ Add another list"
3. Введите название списка
4. В списке нажмите "+ Add a card"
5. Введите название карточки

### 4. Редактирование карточки
1. Кликните на карточку
2. Измените название и/или описание
3. Нажмите "Save"

### 5. Real-time тестирование
1. Откройте доску в двух разных браузерах/вкладках
2. Войдите под разными пользователями
3. Создайте карточку в одной вкладке
4. Она должна автоматически появиться во второй вкладке

### 6. Добавление участников (пока не реализовано в UI)
Можно добавить через API:
```bash
curl -X POST http://localhost:5000/api/boards/{boardId}/members \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"email": "john@example.com"}'
```
